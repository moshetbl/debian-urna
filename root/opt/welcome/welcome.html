Sistema Urna 0.11
O sistema URNA irá configurar agora todos os periféricos necessários.
Antes de continuar, verifique se tudo está corretamente conectado. Para esse sistema é necessário:
  • 2 monitores (um para o mesário e outro para urna)
  • 2 teclados (um para o mesário e outro para urna)
  • 1 dispositivo de armazenamento USB formatado em VFAT
  • 1 impressora

Caso ocorra algum problema, você poderá executar essas configurações novamente através do terminal de configurações,
pressionando a qualquer momento as teclas CTRL+ALT+T

Pressione qualquer tecla para continuar
